##  Featured Projects

### [AeroAtsu — FDR Validation Framework](https://github.com/Ojassv/AeroAtsu)
A modular Python framework automating Flight Data Recorder (FDR) data quality checks for airline onboarding at OpenAirlines. Reduced manual review effort by 50%. Built with Python, pandas, NumPy, Streamlit and Plotly — validates decoded FDR data before ingestion into SkyBreathe®.

### [IoT Sensor Data Pipeline — Medallion Architecture](https://github.com/Ojassv/iot-sensor-pipeline-databricks-project)
End-to-end industrial IoT pipeline on Databricks using Apache Spark, Delta Live Tables, and Unity Catalog. Bronze → Silver → Gold with automated data quality enforcement, rolling z-score anomaly detection, and a 4-page AI/BI monitoring dashboard.

### [Anomaly Detection in Aircraft Systems](https://github.com/Ojassv/Anomaly-Detection)
Unsupervised time-series anomaly detection on multivariate aircraft sensor data using an LSTM Autoencoder (TensorFlow/Keras). Multi-threshold confidence scoring across 1,637 flight windows with per-parameter explainability.
